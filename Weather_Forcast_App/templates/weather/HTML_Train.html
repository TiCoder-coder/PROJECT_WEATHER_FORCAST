📖 Hướng dẫn sử dụng

1

Chọn Dataset

Chọn thư mục dữ liệu và file từ danh sách dropdown. Hoặc cuộn xuống bảng "Datasets có sẵn" và bấm nút "✅ Dùng" để chọn nhanh. Nên sử dụng data đã clean (Cleaned Merge) để có kết quả tốt nhất.

2

Cấu hình Model

Có 2 chế độ cấu hình:

  • ⚡ Cấu hình nhanh: Chọn loại model (XGBoost, Random Forest, LightGBM, CatBoost hoặc Ensemble), điều chỉnh tỷ lệ test/validation
  • 🔧 Cấu hình nâng cao: Chỉnh trực tiếp file JSON config, kiểm soát toàn bộ tham số

Tick "Sử dụng config mặc định" để dùng config từ file train_config.json.

3

Huấn luyện & Xem kết quả

Bấm "🚀 Bắt đầu huấn luyện". Theo dõi tiến trình qua thanh progress bar và nhật ký chi tiết. Khi hoàn tất, kết quả metrics (R², RMSE, MAE) sẽ hiển thị ở mục "Kết quả huấn luyện gần nhất". Model được lưu tại Machine_learning_artifacts/latest/.

💡 Mẹo sử dụng

  • Ensemble (Voting) thường cho kết quả tốt nhất - kết hợp nhiều model
  • Target column mặc định là rain_total - thay đổi nếu cần dự báo biến khác
  • Tỷ lệ Test size = 0.15 và Valid size = 0.15 là cấu hình khuyến nghị
  • Kiểm tra Overfit Status trong Diagnostics — trạng thái "GOOD" là lý tưởng
  • Rain Detection Accuracy cho biết độ chính xác phân loại mưa/không mưa
