📖 Hướng dẫn sử dụng

1

Kiểm tra Model

Đảm bảo đã có model được huấn luyện (hiển thị ở mục "Model hiện tại"). Nếu chưa có, hãy vào trang Huấn luyện ML để train model trước.

2

Chọn phương thức dự báo

Có 3 cách để chạy dự báo:

  • 📦 Từ dataset có sẵn: Chọn thư mục → chọn file → bấm "Dự báo"
  • 📤 Upload file: Kéo thả hoặc chọn file CSV/Excel từ máy tính
  • ✏️ Nhập thủ công: Nhập trực tiếp thông số thời tiết để dự báo 1 điểm
3

Xem kết quả

Kết quả hiển thị bảng preview với cột y_pred là giá trị dự báo. File kết quả đầy đủ lưu tại WeatherForcast/predictions.csv.

💡 Mẹo sử dụng

  • Dùng nút "🔮 Dùng" ở bảng datasets phía dưới để chọn nhanh dataset
  • Đặt số dòng tối đa = 0 để dự báo toàn bộ dữ liệu
  • Khi nhập thủ công, hệ thống tự tính trung bình từ min/max
  • Ngưỡng phân loại mưa (predict_threshold) được tối ưu bằng Optuna — xem "Predict Threshold" trong thông tin model
