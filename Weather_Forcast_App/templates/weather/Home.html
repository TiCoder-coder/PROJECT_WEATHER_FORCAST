API OpenWeather

CHỌN CÁCH THU THẬP DỮ LIỆU

1. Thu thập qua API OpenWeather

  • Lấy dữ liệu thời tiết từ các API quốc tế (nhiệt độ, độ ẩm, gió, mưa, v.v.).
  • Phù hợp để có cái nhìn tổng quát trên toàn quốc.

2. Thu thập qua API của Vrain

  • Lấy dữ liệu lượng mưa trực tiếp từ API (nếu Vrain cung cấp).
  • Ổn định, ít phụ thuộc vào giao diện HTML.

3. Thu thập Vrain bằng Selenium (multi-ID)

  • Sử dụng Selenium điều khiển trình duyệt và đọc giao diện Vrain.
  • Phù hợp khi thu thập nhiều tỉnh/trạm cùng lúc.

4. Thu thập HTML Vrain (phân tích nguồn)

  • Đọc HTML và sử dụng regex để trích xuất dữ liệu lượng mưa cho từng trạm.
  • Phù hợp khi cần tinh chỉnh logic phân tích HTML.

🧭 HƯỚNG DẪN SỬ DỤNG WEATHER HUB

Chọn pipeline thu thập → dữ liệu ra output/ → (tuỳ chọn) merge → (tuỳ chọn) clean → xem/tải ở Bộ dữ liệu.

🎯 Bạn làm được gì trên web?
  • Chạy các pipeline thu thập: OpenWeather, Vrain API, Vrain Selenium, Vrain HTML.
  • Xem danh sách dataset theo nhóm thư mục: raw/output / merged / cleaned.
  • Preview trực tiếp (bảng + phân trang) và Download 1 click.
  • Gộp dữ liệu (Merge) và làm sạch (Clean) ngay trên giao diện (wizard + progress).
🧱 Luồng dữ liệu end-to-end
  1. Crawl → file xuất vào data/data_crawl/.
  2. Merge (tuỳ chọn) → tạo file mới trong data/data_merge/.
  3. Clean (tuỳ chọn) → xuất ra cleaned_data/ theo 2 nhánh:
    • Clean_Data_For_File_Merge/ (clean từ merged)
    • Clean_Data_For_File_Not_Merge/ (clean từ output/raw)
  4. Xem/Tải → vào trang Bộ dữ liệu để preview & download.

GIỚI THIỆU VÀ CÁCH SỬ DỤNG NHANH

Hệ thống Django thu thập → gộp (merge) → làm sạch (clean) → xem trước / tải về dataset thời tiết (CSV/XLSX/JSON/TXT) từ nhiều nguồn.

🎯 Bạn làm được gì trên web?

  • Chạy pipeline thu thập (OpenWeather API, Vrain API, Vrain Selenium/HTML).
  • Xem dataset gần đây theo nhóm thư mục (raw / merged / cleaned).
  • Preview trực tiếp trên web (bảng + phân trang) và Download 1 click.
  • Gộp dữ liệu raw → merged và làm sạch dữ liệu theo 2 nhánh (từ merged hoặc từ raw).

🧱 Kiến trúc (3 layer)

  • 🎨 Presentation: Templates + CSS/JS hiệu ứng thời tiết, modal, wizard.
  • 🧩 Application: Views/URLs điều phối request (crawl/merge/clean/view/download).
  • ⚙️ Data/Processing: Scripts xử lý thật + lưu trữ file & DB.
✅ Dữ liệu chính lưu dạng file (CSV/XLSX), DB SQLite dùng cho Django/dev & lưu tổng hợp nếu cần.

🧭 Luồng dữ liệu end-to-end

  1. Thu thập (Crawl) → file được xuất vào output/ (dữ liệu thô).
  2. Gộp (Merge) → tạo file mới trong Merge_data/ (dữ liệu đã gộp).
  3. Làm sạch (Clean) → xuất ra cleaned_data/ theo 2 nhánh:
    • Clean_Data_For_File_Merge/ (clean từ merged)
    • Clean_Data_For_File_Not_Merge/ (clean từ output/raw)
  4. Xem/Tải → vào mục Bộ dữ liệu để preview & download.

🧩 Các pipeline hiện có

🌐 OpenWeather API 🌧️ Vrain API 🕷️ Vrain Selenium 🧾 Vrain HTML Parser
  • OpenWeather: nhiệt độ/độ ẩm/gió/mây… theo thời gian thực.
  • Vrain: dữ liệu lượng mưa theo trạm (API hoặc crawl theo giao diện).

🚀 Quick start

  1. Nhấn Bắt đầu thu thập data → chọn phương pháp crawl.
  2. Chờ chạy xong → file sẽ xuất hiện trong output/.
  3. Vào Xử lí dữ liệu để 👀 Xem hoặc ⬇️ Tải.
  4. Nếu muốn dữ liệu “chuẩn phân tích”: bấm 🔗 Gộp rồi 🧹 Làm sạch.
