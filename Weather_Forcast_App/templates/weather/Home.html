CHỌN CÁCH THU THẬP DỮ LIỆU

1. Thu thập qua API OpenWeather

  • Lấy dữ liệu thời tiết từ các API quốc tế (nhiệt độ, độ ẩm, gió, mưa, v.v.).
  • Phù hợp để có cái nhìn tổng quát trên toàn quốc.

2. Thu thập qua API của Vrain

  • Lấy dữ liệu lượng mưa trực tiếp từ API (nếu Vrain cung cấp).
  • Ổn định, ít phụ thuộc vào giao diện HTML.

3. Thu thập Vrain bằng Selenium (multi-ID)

  • Sử dụng Selenium điều khiển trình duyệt và đọc giao diện Vrain.
  • Phù hợp khi thu thập nhiều tỉnh/trạm cùng lúc.

4. Thu thập HTML Vrain (phân tích nguồn)

  • Đọc HTML và sử dụng regex để trích xuất dữ liệu lượng mưa cho từng trạm.
  • Phù hợp khi cần tinh chỉnh logic phân tích HTML.

📖 HƯỚNG DẪN SỬ DỤNG

Hướng dẫn từng bước sử dụng các chức năng chính của VN Weather Hub.

🌐 Bước 1 — Thu thập dữ liệu
  1. Từ trang chủ, nhấn 🌐 Bắt đầu thu thập data.
  2. Chọn 1 trong 4 nguồn: OpenWeather API, Vrain API, Vrain Selenium, hoặc Vrain HTML.
  3. Hệ thống tự động chạy script thu thập và hiển thị log thời gian thực.
  4. Khi hoàn tất, file dữ liệu (CSV/Excel) sẽ xuất hiện trong thư mục output/.
📊 Bước 2 — Xử lí dữ liệu
  1. Nhấn 📊 Xử lí dữ liệu để vào trang quản lí dataset.
  2. 👀 Xem: preview bảng dữ liệu trực tiếp (có phân trang). ⬇️ Tải: download file về máy.
  3. 🔗 Gộp: gom nhiều file thành 1 dataset chung (Merge).
  4. 🧹 Làm sạch: chuẩn hoá dữ liệu — xử lý missing values, outliers, normalize, encode…
🧠 Bước 3 — Huấn luyện Machine Learning
  1. Nhấn 🧠 Huấn luyện Model để vào trang training.
  2. Chọn file dataset đã clean, cấu hình model (Ensemble, CatBoost, XGBoost, Random Forest, LightGBM).
  3. Bấm Bắt đầu huấn luyện — theo dõi log và tiến trình real-time.
  4. Xem kết quả đánh giá: R², RMSE, MAE, Rain Accuracy.
🔮 Bước 4 — Dự báo thời tiết
  1. Nhấn 🔮 Dự báo thời tiết để vào trang dự báo.
  2. Dự báo theo file: upload file CSV chứa dữ liệu đầu vào → nhận kết quả hàng loạt.
  3. Dự báo thủ công: nhập từng tham số (nhiệt độ, độ ẩm, gió…) → nhận kết quả ngay.
  4. Kết quả bao gồm lượng mưa dự báo và xác suất mưa.

GIỚI THIỆU HỆ THỐNG

Nền tảng thu thập, xử lí, huấn luyện và dự báo thời tiết Việt Nam — tất cả trong một giao diện.

🌦️ VN Weather Hub là gì?

VN Weather Hub là hệ thống web được xây dựng trên Django, giúp bạn thực hiện toàn bộ quy trình từ thu thập dữ liệu thời tiết đến dự báo lượng mưa bằng Machine Learning. Hệ thống hỗ trợ nhiều nguồn dữ liệu (OpenWeather, Vrain) và nhiều định dạng file (CSV, XLSX, JSON, TXT).

⭐ Tính năng chính

  • 🌐 Thu thập dữ liệu — 4 nguồn: OpenWeather API, Vrain API, Vrain Selenium, Vrain HTML Parser.
  • 📊 Xử lí dữ liệu — xem, tải, gộp (merge) và làm sạch (clean) dữ liệu ngay trên web.
  • 🧠 Huấn luyện ML — hỗ trợ Ensemble, CatBoost, XGBoost, Random Forest, LightGBM.
  • 🔮 Dự báo — dự báo lượng mưa và thời tiết theo file hoặc nhập thủ công.
  • 🔒 Tài khoản — đăng ký, đăng nhập, quên mật khẩu (hỗ trợ OTP qua email).

🔄 Quy trình sử dụng

🌐 Thu thập → 🔗 Gộp → 🧹 Làm sạch → 🧠 Huấn luyện → 🔮 Dự báo
  1. Thu thập: chạy crawl từ 1 trong 4 nguồn → file xuất ra output/.
  2. Gộp (tuỳ chọn): kết hợp nhiều file thành 1 dataset chung.
  3. Làm sạch (tuỳ chọn): xử lý missing values, outliers, chuẩn hoá dữ liệu.
  4. Huấn luyện: chọn dataset đã clean → cấu hình model → train & đánh giá kết quả.
  5. Dự báo: dùng model đã train để dự đoán lượng mưa và thời tiết.
